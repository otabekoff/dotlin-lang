- [io_functions.md](io_functions.md) - Input/Output functions (print, println, readln)

## Language Overview

Dotlin is a programming language that provides native 1:1 syntax compatibility with Kotlin without depending on the JVM. The language is implemented in C++ and compiles to native binaries.

## File Extensions

Dotlin source files use the `.lin` extension.

## Current Status

Dotlin is currently in development with support for:
- Basic variable declarations
- Data types (integers, doubles, booleans, strings)
- Operators (arithmetic, comparison, logical)
- Control flow (if/else statements)
- Functions (including main function)
- I/O operations (print, println, readln)
